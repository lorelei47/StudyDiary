## DOM2和DOM3

#### DOM变化

```
DOM2 级和 3 级的目的在于扩展 DOM API，以满足操作 XML 的所有需求，同时提供更好的错误处理及特性检测能力
```

> 针对XML命名空间的变化

```html
XHTML 的命名空间是 http://www.w3.org/1999/xhtml
<html xmlns="http://www.w3.org/1999/xhtml"> 
    <head> 
        <title>Example XHTML page</title> 
    </head> 
    <body> 
        Hello world! 
    </body> 
</html>
```
